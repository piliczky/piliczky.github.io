This stylised offroad vehicle was created as a game-ready asset, focusing on clean shapes, efficient topology, and a bold silhouette suitable for real-time applications.

The vehicle was modeled and designed in Blender using real-world offroad vehicles as inspiration. The goal was to create a chunky, stylised design that feels both rugged and approachable, while maintaining an efficient triangle count suitable for modern game engines.

### UV Mapping and Texturing

The model was UV unwrapped in Blender using the UniV add-on, allowing efficient use of texture space and consistent texel density across the asset.

Texturing was completed in Substance Painter using a single 1024×1024 texture set. Materials were intentionally kept simple and clean, using colour variation and subtle gradients to reinforce the stylised aesthetic while maintaining readability from a distance.

### Presentation

The final presentation sheet was assembled in Photoshop, combining multiple rendered views, a wireframe render, and the texture layout into a single image. This format provides an overview of the complete asset creation process, from modelling and UV mapping through to texturing and final presentation.

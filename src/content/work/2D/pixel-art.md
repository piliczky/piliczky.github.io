This collection showcases a variety of **pixel art assets**, **sprite animations**, and **game environments** created in **Aseprite**. These personal projects were created to improve my pixel art skills while exploring retro-inspired game development, limited colour palettes, sprite animation, and environment design.

Many of these assets were created using predefined palettes from **Lospec**, including **DawnBringer 16**, **DawnBringer 32**, and several other community-created palettes. Working within strict colour limitations helped improve my understanding of colour harmony, readability, and efficient asset creation for games.

The collection includes animated characters, vehicles, environmental props, complete game levels, sprite sheets, and a monochrome Game Boy-inspired tileset. Several projects demonstrate the complete workflow from sprite sheet creation through to finished looping animations.

### Design Process

Each asset followed a simple workflow:

* **Creating the artwork** directly in **Aseprite**.
* **Building sprite sheets** for animation where required.
* **Animating characters and vehicles** frame-by-frame.
* **Testing assets together** inside complete game environments.
* **Exporting finished animations** as looping GIFs for use in games.

### Summary

While my primary focus is technical art and game development, I enjoy creating **pixel art** to further develop my understanding of sprite animation, limited colour palettes, tile-based environment creation, and efficient game asset production. These projects provided valuable experience creating animated characters, vehicles, environments, and complete game assets using **Aseprite**, while working within the creative constraints of classic retro-inspired colour palettes.

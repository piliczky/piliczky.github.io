This project is a **2D character animation** featuring a stylised robot created in **Adobe Illustrator** and animated in **Adobe After Effects**.

The robot was constructed from individual vector components in Illustrator, allowing each body part to be animated independently. These assets were then imported into After Effects where I rigged and animated the character, producing a smooth animation that loops seamlessly.

### Workflow

The project followed a simple animation pipeline:

* **Designed and illustrated** the robot using vector artwork in **Adobe Illustrator**.
* **Separated individual body parts** to allow independent animation.
* **Imported the artwork into Adobe After Effects**.
* **Animated the robot** using keyframes and transform controls.
* **Created a seamless looping animation** suitable for use in games, websites or motion graphics.

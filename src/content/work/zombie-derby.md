### Project Overview

**Zombie Derby** is a fast-paced action game developed in a team environment. As the **Lead Game Designer and Programmer**, I was responsible for the core gameplay loop, physics implementation for the vehicular combat, and the overall technical architecture of the game systems.

In Zombie Derby, the goal is simple but satisfying: drive a massive monster truck through hordes of zombies. Players collect skulls from fallen enemies which serve as currency to upgrade weapons and truck performance, allowing for longer survival runs and higher scores.

### Technical Implementation

From a programming perspective, I focused on creating a robust physics-based movement system for the truck to ensure that "crushing" felt impactful. I also built the upgrade system using a modular architecture, allowing the team to easily iterate on new weapon types and stat buffs during the development cycle.
